== Description ==

Workaround for user login with caching mechanisms on freistilbox.com hosted websites.

== Installation ==

1. Upload `cached-community.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `cached-community` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the provided JavaScript api in your frontend code
